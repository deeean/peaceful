# peaceful

Peaceful is a simple image processing tool.

## Basic Usage

```bash
$ peaceful convert ./testdata/lenna.png ./testdata/lenna.jpg
```

## Features

| Command | Description | Completed |
| --- | --- | --- |
| convert | Convert image format | ✅ |
| resize | Resize image | ❌ | 
| crop | Crop image | ❌ |
| rotate | Rotate image | ❌ |
| flip | Flip image | ❌ |
| flop | Flop image | ❌ |
| compress | Compress image | ❌ |
